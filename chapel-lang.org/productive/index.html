Safe and Easy Coding

Config Declarations

Language-level support for overriding default values from the command-line, at both compile-time and run-time.

Interoperability

Chapel is designed to interoperate seamlessly with libraries written in C. No need to rewrite important libraries.

Memory Safety

Scope-based (de)allocation, ownership management, default-safe strings, and other semantic checks make Chapel safer than C and C++.
