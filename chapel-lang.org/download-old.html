Chapel version 2.3 was released on December 12, 2024. An overview of its features can be found in its release announcement and its CHANGES.md file.

Pre-packaged Chapel Installations

Pre-packaged Chapel installations are available for:
  • HPE Cray EX and Cray XC systems
  • Linux users via standard package managers
  • Homebrew users on Mac OS X or Linux
  • Spack users
  • Docker users

Building Chapel from Source

All other users will need to download

chapel-2.3.0.tar.gz

from GitHub, then unpack and build it as described in the Quickstart instructions.
