To install and use Chapel on your own machine, take the following steps:

  1. Download version 1.2.0 of the release from http://www.sourceforge.net/projects/chapel

  2. Unpack the release:
    • cd to a directory where you wish to install Chapel as a subdirectory named chapel/
    • gunzip chapel-v1.2.0.tar.gz
    • tar xf chapel-v1.2.0.tar

  3. Read the top-level README to build the compiler, set up your environment, and compile and run your first sample program.

  4. The bottom of the README has possible next steps for exploring the release. For this session, we recommend exploring the examples/ directory, working through the exercises on the previous page, or writing up a program of your own liking; but you're welcome to do whatever appeals to you.

  5. For best results, we recommend using Chapel in a single-locale multithreaded mode. Once you are comfortable with that, you may want to move on to multi-locale Chapel which supports multi-threaded execution across a distributed memory system. See $CHPL_HOME/doc/README.multilocale for instructions to get started.
