Series

Below is a list of all series of articles on this site.

  • 7 Questions for Chapel Users

    This series interviews users of Chapel about their experiences with the language.

  • GPU Programming in Chapel

    This series showcases Chapel’s support for vendor-neutral GPU programming.

  • Navier-Stokes in Chapel

    A series focused on scientific computing in Chapel using steps from the Lorena A. Barba group’s 12 steps to Navier-Stokes tutorial.

  • Advent of Code 2022

    This series is an introduction to the Chapel language through solutions to the first twelve days of Advent of Code 2022.

  • NetCDF in Chapel

    This series describes the use of Chapel’s interoperability features to read NetCDF files into distributed arrays.
