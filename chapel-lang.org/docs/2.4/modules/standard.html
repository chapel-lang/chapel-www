Standard Modules¶

Standard modules are those which describe features that are considered part of the Chapel Standard Library.

Automatic Modules¶

All Chapel programs automatically use the following modules by default:

  • Errors
  • Types
  • A subset of the IO symbols
  • A subset of the Math symbols
