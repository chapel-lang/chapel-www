BinaryIO
========
**Usage**

.. code-block:: chapel

   use BinaryIO;


or

.. code-block:: chapel

   import BinaryIO;


The BinaryIO module provides a BinarySerializer and BinaryDeserializer that
implement a simple binary IO format.


.. warning::

   BinaryIO module is deprecated; please use the IO module instead
